# Racing Macropad

A macropad meant to be used as a buttonbox for simracing. Features 2 led lights, 6 mx-switches and one rotary encoder. The keys act as the numpad, meant for easy binding in popular simracing titles.

## Hardware
*   **MCU:** RP2040
*   **Layout:** 6 keys + 1 Encoder

## Compilation example
`make racing_macropad:default`

## Flashing example
`make racing_macropad:default:flash`
